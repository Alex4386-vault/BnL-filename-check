# BnL-filename-check
가천대학교 소프트웨어학과 2학기 필수 수강 필요한 교양 과목 김원 교수님의 (기업과 리더십)의 파일이름 규격에 맞는지 확인하는 프로그램입니다.  
This program will check your word or powerpoint document meets Prof. Kim won's *high standard* filename format.

Built with  

- TypeScript
- Electron
- REGular EXpression
